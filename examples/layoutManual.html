How to use

If you can interact with the various elements of the layout you can use:

  • this.jq jquery to access the methods of the parent container.
  • this.jqHeader jquery to access the methods of the header.
  • this.jqBody jquery to access the methods of the body.
  • this.jqLeftSidebar jquery to access the methods of the left sidebar.

    • Examples:

    this.jqBody.append('Hello world');
    this.jqBody.load('index.html');

    List property
